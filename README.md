# Memvault Releases

This repository hosts release artifacts for [Memvault](https://github.com/zacostudio/memvault).

Download the latest version from the [Releases](https://github.com/zacostudio/memvault/releases) page.

## Claude Code Plugin

The **memvault-marketplace** plugin lets you manage memvault notes directly from Claude Code.

### Install

In Claude Code, run:

```
/plugin marketplace add zacostudio/memvault.gh
/plugin install memvault-marketplace@zacostudio-memvault-gh
```

See [memvault-marketplace/README.md](./memvault-marketplace/README.md) for details.
